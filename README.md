# Cosmetric AI – Manual QA Testing Project

## Overview
This repository demonstrates manual testing performed on the **Cosmetric AI** web application.  
It contains structured test documentation used to validate core functionality and improve user experience.

**Website tested:** https://cosmetric.ai

## Purpose
The goal was to:
- Identify bugs & usability issues
- Write clear test cases
- Document actual results and test execution

## Testing Types
- Functional Testing  
- UI / UX Testing  
- Exploratory Testing  
- Negative Testing  

## Repository Structure
| File | Description |
|------|-------------|
| `TestPlan.md` | Overall test strategy and scope |
| `TestCases.md` | Structured test cases |
| `BugReports.md` | Documented bugs with reproduction steps |
| `TestExecution.md` | Test results and observations |

## Key Skills Demonstrated
- Requirements analysis
- Test case design
- Bug reporting
- Test execution tracking
